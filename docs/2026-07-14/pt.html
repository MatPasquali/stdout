Read It Back: Pretrained MLLMs Are Zero-Shot Reward Models for Text-to-Image Generation

Pesquisadores apresentaram um método inovador para tornar modelos de linguagem pré-treinados em ferramentas de recompensa para a geração de imagens a partir de textos, utilizando uma abordagem chamada SpectraReward. Isso é importante porque permite que os modelos de linguagem sejam usados para avaliar a qualidade das imagens geradas sem a necessidade de treinamento adicional. A técnica funciona medindo quão bem o texto original pode ser recuperado a partir da imagem gerada, utilizando a probabilidade logarítmica condicionada à imagem como recompensa. Essa abordagem simplifica o processo de geração de imagens e pode ser útil em várias aplicações.

Requential Coding: Pushing the Limits of Model Compression with Self-Generated Training Data

Pesquisadores apresentaram um novo método de compressão de modelos chamado codificação requencial, que visa superar os limites atuais da compressão de modelos com dados de treinamento autogerados. Isso é importante porque a compressão é fundamental para a inteligência, pois um modelo que pode representar seus dados de treinamento como um código curto descobriu regularidades que permitem generalização. A codificação requencial busca contornar as limitações dos métodos baseados em parâmetros, como a quantização, que produzem comprimentos de código que escalam com o tamanho do modelo, e em vez disso, selecciona amostras de treinamento com base na distribuição do modelo estudante.
