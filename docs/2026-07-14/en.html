Requential Coding: Pushing the Limits of Model Compression with Self-Generated Training Data

Researchers have introduced a new model compression method called requential coding, which aims to push the limits of model compression with self-generated training data. This matters because compression is fundamental to intelligence, as a model that can represent its training data as a short code has discovered regularities that enable generalization. Requential coding bypasses the limitations of parameter-based methods, such as quantization, which produce code lengths that scale with model size, and instead selects training samples based on the model's own distribution. By doing so, it addresses the issue of large codes being generated when the data has high entropy.
