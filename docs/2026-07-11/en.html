Ideas Have Genomes: Benchmarking Scientific Lineage Reasoning and Lineage-Grounded Idea Generation

This matters because scientific ideas rarely start from scratch, but instead build upon existing mechanisms, fix known limitations, and recombine parts of earlier work. The IG-Bench is organized around the IdeaGene framework, which represents each paper or proposal as a set of minimal, typed, evidence-grounded Idea Genome objects. SLORR: Simple and Efficient In-Training Low-Rank Regularization

Researchers have introduced a new method called SLORR, which aims to make neural network compression more efficient through in-training low-rank regularization, a crucial technique for reducing model size without losing accuracy. Low-rank regularization is a method that seeks to find simpler weight matrices, but current methods often require complex calculations or modify the model architecture. SLORR is a simple and efficient approach that can be applied directly to the original weight matrices, without changing the model architecture, and is based on metrics such as the Hoyer sparsity metric and the nuclear norm.
