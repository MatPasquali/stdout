LeVo 2: Stable and Melodious Song Generation via Hierarchical Representation Modeling and Progressive Post-Training

Pesquisadores apresentaram o LeVo 2, um modelo híbrido de linguagem e difusão para geração de músicas completas, que busca superar os limites dos sistemas atuais, que enfrentam um dilema entre preservar a coordenação entre vocais e instrumentos e manter a clareza dos detalhes de cada faixa. O LeVo 2 utiliza uma abordagem hierárquica, primeiro prevendo tokens mistos para planejamento semântico, e então prevendo tokens de vocais e acompanhamento em paralelo para refinar os detalhes de cada faixa, enquanto um codificador de música baseado em difusão reconstrói as ondas sonoras completas. Isso permite uma geração de músicas mais controlável e de alta qualidade. O modelo pode ser útil para aplicações de criação de música assistida por computador.

GROW$^2$: Grounding Which and Where for Robot Tool Use

Pesquisadores publicaram um artigo sobre GROW$^2$, uma abordagem que permite que robôs usem objetos de forma criativa como ferramentas, mesmo que não sejam os objetos tradicionalmente utilizados para essa tarefa, como usar um prato para cortar um bolo se não houver faca disponível. Isso é possível graças à técnica de "grounding de affordance em mundo aberto", que consiste em selecionar um objeto de categoria aberta para atuar como ferramenta e localizar sua região específica de ação. A abordagem GROW$^2$ divide o processo de grounding de forma hierárquica em níveis semânticos e geométricos, utilizando partes de objetos como abstração natural.
