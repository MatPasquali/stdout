Paying More Attention to Visual Tokens in Self-Evolving Large Multimodal Models

Recently, self-evolving large multimodal models (LMMs) have received attention for improving visual reasoning in a purely unsupervised setting. However, the multi-role self-play and self-consistency reward schemes in existing self-evolving LMMs optimize answer agreement without ensuring the decoder attends to visual content, relying instead on statistical language priors to produce self-consistent outputs, leading to a persistent failure mode known as visual under-conditioning, where the decoder relies on language priors rather than the image during generation. To address this, researchers propose VISE, an approach that aims to improve attention to visual tokens in self-evolving LMMs. This is significant because current self-evolving LMMs struggle with vision-language understanding tasks such as image captioning and visual question answering.

Reinforcement Learning without Ground-Truth Solutions can Improve LLMs

A recent study published on arXiv found that reinforcement learning without ground-truth solutions can improve large language models, as current training methods for these models typically rely on correct answers to assign rewards, limiting their applicability to tasks where the correct solution is unknown. The researchers introduced a framework called RiVER, which trains large language models on score-based optimization tasks without ground-truth solutions, using deterministic execution feedback as continuous-valued supervision. This can be especially useful in tasks where the correct answer is unknown or difficult to obtain, and it addresses key challenges such as scale dominance and frequency dominance that arise when using continuous rewards.
