### [AutoMem: Automated Learning of Memory as a Cognitive Skill](https://arxiv.org/abs/2607.01224v1)
`arXiv`

Researchers have developed AutoMem, an approach that enables large language models to learn memory management as a cognitive skill, making the learning and decision-making process more efficient. This matters because memory is a learned skill that involves knowing what to encode, when to retrieve, and how to organize knowledge, known as metamemory in cognitive science. With AutoMem, models can decide how to manage their own memory, improving their ability to perform long-term tasks. This is achieved by promoting file-system operations to first-class memory actions, allowing the model to learn to manage its memory autonomously.

### [Ink3D: Sculpting 3D Assets with Extremely Complex Textures via Video Generative Models](https://arxiv.org/abs/2607.01222v1)
`arXiv`

Researchers have introduced Ink3D, a new approach to creating 3D assets with extremely complex textures, using video generative models that are trained on large datasets and excel at modeling complex visual patterns. This matters because current 3D generative models struggle to reproduce intricate textures from reference images, largely due to the scarcity of large-scale 3D training data with rich surface appearance. Ink3D works by first reconstructing a white-mesh geometry using an off-the-shelf 3D generation model, and then employing OrbitPainter, a conditional video generative model, to produce dense orbit-scan videos that capture object appearance across viewpoints. This approach enables the creation of extremely complex textures for 3D assets.
